# 📦 Vault 구조 정리법

> 이 문서는 본 Obsidian Vault의 전체 폴더 구성과 각 폴더의 용도를 정리한 가이드입니다.  
> 링크 기반 PKM, 실험·연구·보고서 작성 중심으로 설계되어 있습니다.

## 🗂️ 폴더 구조 설명

📁 0.INBOX/ 
	임시 메모, 아이디어, 클리핑 등  
📁 1.DAILY NOTES/ 
	일일 일지, 실험 로그, 업무 흐름
📁 2.REVIEWS/ 
	논문/서적 리뷰 노트  
📁 3.CONCEPTS/ 
	개념 정리, 모델 정의, 용어 설명  
📁 4.IDEAS/ 
	“이건 나중에 어디든 쓸 수 있을 아이디어다”
📁 5.LAB/ 
	실험/시뮬레이션 수행 기록  
📁 6.PROJECTS/ 
	실제 연구/졸업논문/과제별 노트  
	“이건 프로젝트만을 위한 실험 계획/아이디어다”
└─ 졸업논문/ # 졸업논문 관련 정리  
└─ 프로젝트A/ # 프로젝트별 정리  
└─ 프로젝트B/
📁 7.OUTPUTS/ 
	보고서, 발표자료, 논문 초안, MOC 등
📁 97.TEMPLATES/ 
	템플릿 모음 (Templater, QuickAdd 등)  
📁 98.RESOURCES/ 
	PDF, 참고문헌, 리포트 등 지식성 자료  
📁 99.ATTACHMENTS/ 
	이미지, 도식, 스크린샷 등 시각 자료
